Once I am a user how do I access MIPMIS?

You can access the Log In page to MIPMIS via the website www.MIPMIS.co.za, which have a Login option to MIPMIS.

How do I Log In to MIPMIS?

a)      Select the Login option from the www.MIPMIS.co.za website.

b)      The system displays the Login screen.

c)      Enter the required User Name and Password.

d)      The User Name and Password is case sensitive.

e)      Click the Submit button.

f)       The system displays your Personalised Homepage.

How do I Navigate within MIPMIS?

a)      The Top Menu Bar allows you to navigate between the different MIPMIS Modules.

b)      Breadcrumbs keeps track of your navigation and by clicking on a screen name in the list you can return to that screen e.g. Project Name>Risks>Risk Description

c)      Hyperlinks open more information about the subject e.g. by clicking on a Project Number the Project Overview screen will open. Hyperlinks is all indicated in blue text.

d)      Icons allows you to access sub-modules or edit a record e.g. to capture a project’s Issues you can click on the flame icon. You can hover over the icon with your mouse pointer and it will display a tooltip of its function.                                                                                                

How do I Add, Edit or Delete a Record?     

a)      Add a new Record by clicking on the Plus icon.

b)      Edit a Record by clicking on the Pen icon.

c)      Only the MIPMIS Administrators are allowed to Delete a Record. You can contact the Administrator via e-mail, administrator@mipmis.co.za.

d)      Please ensure that you cannot use the edit function to make the corrections before contacting the Administrator.

e)      MIPMIS keeps a full audit trail of your record addition, edits and deletions.

Where do I find the Reports and Dashboards?        

All the MIPMIS Reports and Dashboards is located in the Insights module.
